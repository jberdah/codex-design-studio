# Devpost image gallery

These six 1800×1200 PNG files are generated from real Codex Design Studio renders with:

```bash
npm run media:devpost
```

The gallery follows a single story:

1. product promise;
2. evidence-aware bootstrap;
3. versioned Design System;
4. real artifact editing;
5. transactional visual QA; and
6. shared editable outputs.

Before regenerating, make sure `projects/demo` contains the fictional Asteria demo state intended for the submission. The generator opens only the local application, uses a non-routable example URL in the unsubmitted bootstrap form, and removes raw screenshots after composition.
